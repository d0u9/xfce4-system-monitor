# How to use

## Prerequisites

For Debian or Ubuntu users.

```
sudo apt-get install xfce4-panel-dev libgtk2.0-dev libxfce4ui-1-dev
```

## Build and install

```
sudo make all
```

## Uninstall

```
sudo make uninstall
```

## Tested Platform

Have tested on Debian8.
